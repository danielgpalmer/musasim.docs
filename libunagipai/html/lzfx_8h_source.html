72 /* Buffer-to-buffer decompression.
73 
74  Supply pre-allocated input and output buffers via ibuf and obuf, and
75  their size in bytes via ilen and olen. Buffers may not overlap.
76 
77  On success, the function returns a non-negative value and the argument
78  olen contains the uncompressed size in bytes. On failure, a negative
79  value is returned.
80 
81  If the failure code is LZFX_ESIZE, olen contains the minimum buffer size
82  required to hold the decompressed data. Otherwise, olen is not modified.
83 
84  Supplying a zero *olen is a valid and supported strategy to determine the
85  required buffer size. This does not require decompression of the entire
86  stream and is consequently very fast. Argument obuf may be NULL in
87  this case only.
88  */
