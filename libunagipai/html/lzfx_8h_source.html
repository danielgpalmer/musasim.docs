00061 /*  Buffer-to buffer compression.
00062 
00063  Supply pre-allocated input and output buffers via ibuf and obuf, and
00064  their size in bytes via ilen and olen.  Buffers may not overlap.
00065 
00066  On success, the function returns a non-negative value and the argument
00067  olen contains the compressed size in bytes.  On failure, a negative
00068  value is returned and olen is not modified.
00069  */

00072 /*  Buffer-to-buffer decompression.
00073 
00074  Supply pre-allocated input and output buffers via ibuf and obuf, and
00075  their size in bytes via ilen and olen.  Buffers may not overlap.
00076 
00077  On success, the function returns a non-negative value and the argument
00078  olen contains the uncompressed size in bytes.  On failure, a negative
00079  value is returned.
00080 
00081  If the failure code is LZFX_ESIZE, olen contains the minimum buffer size
00082  required to hold the decompressed data.  Otherwise, olen is not modified.
00083 
00084  Supplying a zero *olen is a valid and supported strategy to determine the
00085  required buffer size.  This does not require decompression of the entire
00086  stream and is consequently very fast.  Argument obuf may be NULL in
00087  this case only.
00088  */
