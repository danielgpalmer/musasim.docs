Detailed Description

A collection of common DMA operations.

Date:
Apr 10, 2012
Author:
daniel

A collection of common DMA operations for example block fills, block copies, block to register (for copying blocks to fifos etc), and register to block (for reading blocks from fifos etc). For more complex operations program the DMA controller directly.
